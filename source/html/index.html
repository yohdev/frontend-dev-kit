Hello world! This is pace-prototype.

Consult the readme on how to use this kit.

This uses Bootstrap 4.4.1 as a framework along with HTML partials through gulp-include for a bare-bones prototyping kit.

We include the head and body tags for maximum control while building (ex. templating classes on the body element, or page specific css or js)

See about/index.html for an example of subpage setup.
